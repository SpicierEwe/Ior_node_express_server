Employee Activities API Documentation

Welcome to the API documentation for the Employee Activities Management System. Below you'll find details on how to use each endpoint.

GET /

Returns this very documentation.

curl -X GET http://localhost:3000/

GET /activities

Fetches all employee activities.

curl -X GET http://localhost:3000/activities

GET /activities/:id

Fetches a specific employee activity by ID.

curl -X GET http://localhost:3000/activities/3

DELETE /activities/:id

Deletes an employee activity by ID.

curl -X DELETE http://localhost:3000/activities/3

Database Schema

The table structure for the emp_activity table:

CREATE TABLE emp_activity (
  id SERIAL PRIMARY KEY,
  emp_id UUID NOT NULL,
  employee_activity TEXT NOT NULL,
  leave_id UUID,
  task_id UUID,
  proj_id UUID,
  invoice_id UUID,
  ticket_id UUID,
  proposal_id UUID,
  estimate_id UUID,
  deal_id UUID,
  deal_followup_id UUID,
  client_id UUID,
  expenses_id UUID,
  timelog_id UUID,
  event_id UUID,
  product_id UUID,
  credit_note_id UUID,
  payment_id UUID,
  order_id UUID,
  contract_id UUID,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
