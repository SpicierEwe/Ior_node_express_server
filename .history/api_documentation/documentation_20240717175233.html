Employees API Documentation

Welcome to the API documentation for the Employees Management System. Below you'll find details on how to use each endpoint.

GET /

Returns this very documentation.

curl -X GET http://localhost:3000/

GET /employees

Fetches all employees.

curl -X GET http://localhost:3000/employees

GET /employees/:id

Fetches a specific employee by ID.

curl -X GET http://localhost:3000/employees/3

DELETE /employees/:id

Deletes an employee by ID.

curl -X DELETE http://localhost:3000/employees/3

Database Schema

The table structure for the employees table:

CREATE TABLE employees (
  id INT AUTO_INCREMENT PRIMARY KEY,
  company_id INT,
  user_id INT,
  location_id INT,
  clock_in_time DATETIME,
  clock_out_time DATETIME,
  clock_in_ip VARCHAR(45),
