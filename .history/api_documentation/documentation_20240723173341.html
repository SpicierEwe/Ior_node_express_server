Projects API Documentation

Welcome to the API documentation for the Projects Management System. Below you'll find details on how to use each endpoint.

GET /

Returns this very documentation.

curl -X GET http://localhost:3000/

POST /projects

Creates a new project.

Request Body:

{
    "title": "Project Title",
    "milestone": "Milestone Name",
    "priority": "High",
    "status": "Pending",
    "assign_users": [1, 2],
    "description": "Project Description",
    "more_information": "Additional Information",
    "options": "Options Data",
    "createdAt": "2024-07-01 08:00:00",
    "updatedAt": "2024-07-01 08:00:00"
}
curl -X POST http://localhost:3000/projects -H "Content-Type: application/json" -d '{ "title": "Project Title", "milestone": "Milestone Name", "priority": "High", "status": "Pending", "assign_users": [1, 2], "description": "Project Description", "more_information": "Additional Information", "options": "Options Data", "createdAt": "2024-07-01 08:00:00", "updatedAt": "2024-07-01 08:00:00" }'

GET /projects

Fetches all projects.

curl -X GET http://localhost:3000/projects

GET /projects/:id

Fetches a specific project by ID.

curl -X GET http://localhost:3000/projects/3

PUT /projects/:id

Updates an existing project by ID.

Request Body:

{
    "title": "Updated Project Title",
    "milestone": "Updated Milestone Name",
    "priority": "Medium",
    "status": "In Progress",
    "assign_users": [1, 3],
    "description": "Updated Project Description",
    "more_information": "Updated Additional Information",
    "options": "Updated Options Data",
    "createdAt": "2024-07-01 08:00:00",
    "updatedAt": "2024-07-02 08:00:00"
}
curl -X PUT http://localhost:3000/projects/3 -H "Content-Type: application/json" -d '{ "title": "Updated Project Title", "milestone": "Updated Milestone Name", "priority": "Medium", "status": "In Progress", "assign_users": [1, 3], "description": "Updated Project Description", "more_information": "Updated Additional Information", "options": "Updated Options Data", "createdAt": "2024-07-01 08:00:00", "updatedAt": "2024-07-02 08:00:00" }'

DELETE /projects/:id

Deletes a project by ID.

curl -X DELETE http://localhost:3000/projects/3

Database Schema

The table structure for the projects table:

CREATE TABLE projects (
  id INT AUTO_INCREMENT PRIMARY KEY,
  title VARCHAR(255),
  milestone VARCHAR(255),
  priority VARCHAR(50),
  status VARCHAR(50),
  assign_users JSON,
  description TEXT,
  more_information TEXT,
  options TEXT,
  createdAt DATETIME,
  updatedAt DATETIME
);
